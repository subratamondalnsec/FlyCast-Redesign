# FlyCast Drone Website Implementation

## Overview
This is a modern, responsive drone website built with React, Vite, and TailwindCSS. The design matches the provided mockup with smooth animations and professional styling.

## Components Created

### 1. **Navbar** (`src/components/Navbar.jsx`)
- Clean navigation with logo, menu items, and call-to-action button
- Responsive design with hover effects
- User avatar placeholder

### 2. **HeroSection** (`src/components/HeroSection.jsx`)
- Large hero banner with compelling headline
- Animated drone visualization
- Call-to-action button and feature icons
- Floating statistics card

### 3. **Statistics** (`src/components/Statistics.jsx`)
- Two-column layout with descriptive content and stats
- Interactive action buttons (Scanning, Pest Detection, Spraying)
- Performance metrics grid
- Drone visualization with grass effects

### 4. **DroneModels** (`src/components/DroneModels.jsx`)
- Grid layout showcasing 4 drone models
- Pricing information and features
- Hover effects and call-to-action buttons
- Models: Hecita One, Hecita Scout, Hecita Pro, Hecita Mini

### 5. **FieldInsights** (`src/components/FieldInsights.jsx`)
- Dark theme section with contrasting design
- Feature highlights and statistics
- Interactive cards showing setup time and optimization
- Bottom statistics grid

### 6. **Footer** (`src/components/Footer.jsx`)
- Simple footer with navigation links
- Legal links and copyright information

### 7. **AnimatedSection** (`src/components/AnimatedSection.jsx`)
- Custom animation component using Intersection Observer
- Smooth fade-in and slide-up effects
- Configurable delays for staggered animations
